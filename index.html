gencDNA finder for PacBio CCS DNA sequencing

This Python library contains code for detecting genomic cDNA, aka gencDNA from the publication Lee et al. Nature 2018.

Installation

You need to have Python with Poetry installed. See Poetry Installation instructions. Then run this:

git clone https://github.com/igor-sb/gencdna.git
cd gencdna
make install

This will create an indentical Python virtual environment as what was used to create this code and greatly minimize possible errors.

Running the code

There are couple of ways to use the code. From Python for example:

from gencdna.file_api.expected_error_filter import filter_reads_with_low_expected_errors

filter_reads_with_low_expected_errors(
   input_fastq_file,
   output_fastq_file,
   maximum_expected_errors = 0.01,
)

… or from the command line:

python gencdna/file_api/expected_error_filter.py \
   input.fastq \
   output.fastq \
   0.01
